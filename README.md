## 📖 Overview

**LuxeHaven Realty** is a state-of-the-art proptech platform engineered to remove friction, confusion, and mistrust from the real estate discovery journey. Designed to rival global leaders while setting a new standard for UI/UX refinement, LuxeHaven treats property search like luxury e-commerce.

> **Workspace Architecture:** This repository is designed as a scalable workspace. The core frontend web application is located in the [`stellar-website-build-main`](./stellar-website-build-main) directory.

---

## ✨ Key Features

- **Editorial-Grade Discovery:** A highly visual, clutter-free property exploration experience featuring full-screen galleries and seamless transitions.
- **Radical Transparency:** Itemized price breakdowns (Base price, GST, Stamp Duty, Maintenance) with zero hidden fees.
- **Verified Trust Signals:** Built-in verification logic, authentic owner badges, and strictly curated agent profiles.
- **Enterprise Performance:** Built with TanStack Start, ensuring lightning-fast load times, exceptional mobile responsiveness, and world-class SEO architecture.

---

## 🏗️ Architecture & Tech Stack

LuxeHaven is built utilizing a modern, type-safe, and highly scalable frontend architecture:

- **Framework:** [TanStack Start](https://tanstack.com/start) / React 18
- **Routing:** TanStack Router (File-based, fully type-safe)
- **Styling:** Tailwind CSS + Custom CSS Variables (Premium minimal design system)
- **Animation:** Framer Motion (Smooth, 60fps micro-interactions and parallax effects)
- **State Management:** TanStack Query (React Query)
- **Tooling:** Vite, TypeScript, ESLint, Prettier

## 🚀 Getting Started

To run the platform locally, follow these steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Navigate to the core web application package:
   ```bash
   cd stellar-website-build-main
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
